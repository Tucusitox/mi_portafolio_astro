(SGP) Sistema de Gestión de Peticiones

El sistema administra y gestiona las solicitudes o incidencias recibidas por correo electrónico. Para ello, se conecta a un servidor de correo o uan cuenta con clave de aplicación mediante el protocolo IMAP y realiza un análisis automatizado de los mensajes enviados durante el día. Cada correo válido genera un ticket, que se asigna automáticamente a un analista disponible en el sistema. Este usuario recibe una notificación para dar seguimiento o resolver la incidencia, informando al cliente por correo cuando su solicitud ha sido atendida. La aplicación incorpora control de acceso con roles y permisos, protección de rutas, respaldo de la base de datos, operaciones CRUD esenciales y notificaciones por correo electrónico a través del protocolo SMTP.

(SGATI) Sistema de Gestión de Activos TI

Está diseñado para gestionar de manera eficiente los equipos tecnológicos de una empresa, incluyendo computadoras, mouse, monitores, teclados y otros dispositivos periféricos. Centraliza toda la información en un único sistema, facilitando el control de los activos. Además, permite asignar equipos a empleados de manera dinámica y generar reportes en excel o pdf para un mejor análisis de los activos. Cuenta con roles de usuario y protección de rutas, el software garantiza que solo el personal autorizado pueda acceder a funciones críticas, manteniendo la integridad y confidencialidad de los datos.

(CMS) Sistema de Gestión de Contenidos

Este CMS ofrece gestión web intuitiva para empresas, con módulos clave (inicio, noticias y contacto) accesibles desde un panel administrativo. Implementa tres roles de usuario: administrador (acceso total), publicador (borrado lógico) y visitante (solo lectura). El borrado físico requiere contraseña de administrador. Incluye protección de rutas, autenticación por tokens via correo electrónico y recuperación de contraseñas. Garantiza seguridad con respaldos automáticos de base datos y validación de cuentas. Ideal para usuarios sin conocimientos técnicos que necesitan control completo sobre su sitio web.

(SDI) Sistema de Inventarios

Se encarga de la gestión centralizada de todos los productos de un negocio, sin importar su variedad temática, categorías o rubro comercial, ofreciendo un control unificado tanto del inventario como de los proveedores asociados y los usuarios del sistema. Permite administrar de manera eficiente múltiples aspectos desde la organización jerárquica de productos por categorías, hasta el registro detallado de proveedores, pasando por la gestión de usuarios. La plataforma incluye robustas medidas de seguridad, como protección de rutas mediante autenticación y autorización, garantizando así el resguardo de la información sensible.
